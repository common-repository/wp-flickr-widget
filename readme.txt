== Description ==

Widget in the sidebar shows a few of your photos from Flickr.

Features:

*   Display of photographs of a particular user or group.
*   Display random images, or display of the last photos.
*   Adjustable number of images: from 1 to 10.
*   Adjustable CSS class wrappers pictures.

Возможности:

*   Показ фотографий отдельного пользователя или группы.
*   Показ случайных фотографий или показ последних фотографий.
*   Регулируемое количество изображений: от 1 до 10.
*   Регулируемый CSS класс обертки картинок.

== Installation ==

1. Unzip files.
2. Upload 'wp-flickr-widget' to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
